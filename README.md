## <a id="0-프로젝트-소개 a"></a> 0. 프로젝트 소개
### 배경
본 프로젝트는 신규 입사자와 임직원이 인사·복지·근태·교육·사내 제도 관련 정보를 하나의 검색창에서 확인하고, 검색으로 해결되지 않는 문의는 HR 티켓으로 전환하는 인사 업무 지원 플랫폼이다.

처리 완료된 HR 답변은 FAQ나 온보딩 문서로 다시 축적되어 이후 유사 문의에 재사용된다. 즉, 단순 게시판이나 FAQ 시스템이 아니라 *검색 → 문의 처리 → 지식 축적 → 재검색* 구조를 가진 HR 지식 관리 시스템이다.

### 문제 정의
현재 인사·온보딩 정보는 메일, 메신저, 사내 게시판, PDF 매뉴얼, 구두 안내 등 여러 채널에 흩어져 있다.

이로 인해 신규 입사자는 필요한 정보를 찾기 어렵고, HR 담당자는 연차, 근태, 복지, 교육, 장비 지급 등 반복적인 질문에 계속 답변해야 한다.

또한 부서별 온보딩 방식이 달라 신규 입사자의 적응 경험에 차이가 생기며, 기존 문의 처리 이력이 체계적으로 축적되지 않아 유사 문의가 반복되는 문제가 있다.
