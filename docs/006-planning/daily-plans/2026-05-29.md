## 3. 팀원이 "나 오늘 뭐하면 돼?"라고 물었을 때

아래 순서로 안내한다.

```text
1. docs/006-planning/daily-plans/YYYY-MM-DD.md 확인
2. 본인 member-wbs 확인
3. 이번 주 weekly-wbs 확인
4. 오늘 작업 종료 전 본인 daily report 작성
```

예시:

```text
이슬이는 해당 날짜의 daily plan에서 본인 행을 먼저 보고,
member-wbs/lee-seuli.md와 weekly-wbs/2026-06-01-week1.md를 확인한 뒤

황희수는 FE skeleton 전에 Figma Make 화면이 최신 기획과 맞는지 먼저 확인한다.
```
