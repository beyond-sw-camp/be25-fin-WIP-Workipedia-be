## 체크리스트

- [ ] 관리자 대시보드 기본 지표가 조회된다.
- [ ] 관리자 작업이 admin_logs에 기록된다.
- [ ] TEAM_ADMIN은 자기 팀 티켓 큐를 조회한다.
- [ ] SYSTEM_ADMIN은 공통 접수 큐를 조회하고 수동 배정한다.
- [ ] 지식화 후보 검수 상태가 변경된다.
- [ ] 질문/답변/채택 포인트 이력이 남는다.
- [ ] 사용자별 포인트 조회가 동작한다.
- [ ] 랭킹 또는 포인트 요약이 조회된다.
- [ ] ESG 점수 기준 등급 산정이 동작한다.
- [ ] ESG 지표 카드가 조회된다.
- [ ] 프론트 관리자 화면과 연동된다.
