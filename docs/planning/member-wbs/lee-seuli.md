## 체크리스트

- [ ] 회원가입이 동작한다.
- [ ] 로그인이 JWT를 발급한다.
- [ ] USER/TEAM_ADMIN/SYSTEM_ADMIN 권한이 구분된다.
- [ ] 비활성 사용자는 로그인할 수 없다.
- [ ] 챗봇 세션 생성이 동작한다.
- [ ] 챗봇 세션 목록 조회가 동작한다.
- [ ] 사용자 메시지 저장이 동작한다.
- [ ] 챗봇 응답이 메시지 이력에 저장된다.
- [ ] 사용자별 대화 조회가 동작한다.
- [ ] 챗봇 메시지의 `next_action`으로 워키/티켓 전환 흐름을 연결한다.
- [ ] 알림 생성/조회/읽음 API가 동작한다.
- [ ] SSE 기반 실시간 알림 또는 폴링 fallback이 동작한다.
- [ ] 답변/채택/티켓 상태 변경/이관/지식화/포인트/ESG 등급 알림이 생성된다.
