## 체크리스트

- [ ] 회원가입이 동작한다.
- [ ] 로그인이 JWT를 발급한다.
- [ ] USER/ADMIN 권한이 구분된다.
- [ ] 비활성 사용자는 로그인할 수 없다.
- [ ] 챗봇 세션 생성이 동작한다.
- [ ] 사용자 메시지 저장이 동작한다.
- [ ] 챗봇 응답이 메시지 이력에 저장된다.
- [ ] 사용자별 대화 조회가 동작한다.
