## 체크리스트

- [ ] 회원가입이 동작한다.
- [ ] 로그인이 JWT를 발급한다.
- [ ] USER/TEAM_ADMIN/SYSTEM_ADMIN 권한이 구분된다.
- [ ] 비활성 사용자는 로그인할 수 없다.
- [ ] 관리자 사용자 조회·상태·권한 변경이 동작한다.
- [ ] 포인트 이력·요약·랭킹이 조회된다.
- [ ] ESG 등급과 지표 카드가 조회된다.
- [ ] 알림 생성/조회/읽음 API가 동작한다.
- [ ] SSE 기반 실시간 알림 또는 폴링 fallback이 동작한다.
- [ ] 답변/채택/티켓 상태 변경/이관/지식화/포인트/ESG 등급 알림이 생성된다.
