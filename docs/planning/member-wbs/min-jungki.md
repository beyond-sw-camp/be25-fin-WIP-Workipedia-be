## 체크리스트

- [ ] 워키 질문 등록/목록/상세/수정이 동작한다.
- [ ] 답변 등록과 채택이 동작한다.
- [ ] 채택 후 추가 답변 제한이 적용된다.
- [ ] 좋아요/싫어요 반응이 동작한다.
- [ ] FAQ 기본 조회가 동작한다.
- [ ] 프론트 연동용 API 문서가 정리된다.
- [ ] Elasticsearch 컨테이너가 docker-compose에 추가된다.
- [ ] 매뉴얼/워키 chunk가 Elasticsearch에 인덱싱된다.
- [ ] 유사도 검색 adapter가 김진혁 RAG 흐름에 연결된다.
- [ ] 챗봇 화면에서 메시지/출처/워키·티켓 전환 UI가 동작한다.
- [ ] 모바일 화면에서 핵심 시나리오가 깨지지 않는다.
- [ ] PWA/Capacitor 래핑 범위를 정리한다.
- [ ] CDN 챗봇 컴포넌트 주입을 위한 허용 도메인 정책을 백엔드와 맞춘다.
